About Me

I am a 19 year old Software developer based out of Winston-Salem North Carolina. I specialize in HTML, and CSS and can bring any idea you have to life. I learned how to code at Coder Foundry's technical bootcamp, and can create custom webpages using bootsrap.
Download Resume

What i use

I can use c# on a daily basis to build full stack webapplications. You can see my work implementing c# in my projects section!

I use proven design patterns and frameworks to build websites. ASP.NET core web and MVC fits the bill for most projects

I use HTML, CSS and Bootstrap to build beautiful responsive websites. Bootstrap is my go to CSS framework. Many shy away from the front-end, but I embrace it.

Javascript is the language of the web. I use vanilla JS and am comfortable with many JS frameworks. Check out my challenges section for some JS projects!

SQL is the language of the database, My projects all utilize a database. I have used Postgres on all of them. I can write SQL on any DB platform.

I use git and github for all of my projects. Git is the most used source control framework on the planet. I also employ CI/CD for publishind.
